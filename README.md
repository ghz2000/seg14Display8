# seg14Display8

*Ghz2000製 14セグ表示器 のサンプルソースコードです｡*

これはArduino用のサンプルスケッチになっています｡  
ライブラリ化してあるので､そこだけインポートして使用することが出来ます｡

動作確認済み｡ 記号も出せます｡

一度に読み込めるテキストは30文字です｡
それを超えると変な文字が表示されます｡
